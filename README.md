# repositorio_atividade
Repositório para a atividade de git e github 

### Atividade:

1. Crie um novo repositório(marque a opção readme)
2. clonar o repositorio na pasta documentos
3. criar qualquer codigo em JS ou TS
4. utilizar os comandos:
- git status
- git branch
- git checkout
- git add .
- git commit -m
- git push origin “nome_da_branch”
5. Verifique se o arquivo criado foi enviado para uma nova branch. ATENÇÂO: não de push diretamente na main
